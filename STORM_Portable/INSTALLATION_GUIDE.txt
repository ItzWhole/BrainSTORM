üöÄ QUICK SETUP (if WSL2 not installed):

Step 1: Install WSL2
-------------------
1. Open PowerShell as Administrator
2. Run: wsl --install -d Ubuntu-20.04
3. Restart computer when prompted
4. Set up Ubuntu username/password when first launched

Step 2: Install NVIDIA Drivers (Optional but Recommended)
--------------------------------------------------------
1. Visit: https://www.nvidia.com/drivers/
2. Download and install latest drivers
3. Restart computer

Step 3: Setup Dependencies
-------------------------
1. Double-click "setup_dependencies.bat"
2. Wait for installation to complete (5-10 minutes)

üéÆ RUNNING THE APPLICATION:

Option A: Interactive Launcher (Recommended)
------------------------------------------
1. Double-click "STORM_Launcher.bat"
2. Choose option [1] to launch
3. The launcher will check your system automatically

Option B: Direct Launch
----------------------
1. Double-click "launch_storm.bat"
2. Application will start directly

üîß TROUBLESHOOTING:

Problem: "WSL not found"
Solution: Install WSL2 using Step 1 above

Problem: "Ubuntu not found"
Solution: Run "wsl --install -d Ubuntu-20.04" in PowerShell

Problem: "No GPU detected"
Solution: Install NVIDIA drivers from nvidia.com/drivers

Problem: Application won't start
Solution: Run setup_dependencies.bat first

Problem: Slow performance
Solution: Ensure NVIDIA drivers are installed for GPU acceleration
